# A Quick React and Tailwind CSS Starter.

## Current Version <br/>

`React-17.0.2` <br/>
`Tailwind-2.1.2`

## Run the development server:

`1. git clone https://github.com/faysalchowdhury99/react-tailwindcss-boilerplate.git`

`2. cd react-tailwindcss-boilerplate`

`3. yarn`

`4. yarn start`

## Build

`yarn build`
